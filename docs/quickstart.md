# Быстрый старт

Для запуска парсера **ничего устанавливать не нужно**.

### Пошаговая инструкция

1. Скачайте архив с панелью через бота.
2. Распакуйте архив в любую удобную папку.
3. Запустите файл `csparser.exe`.
4. Скопируйте HWID полученый в консоли
5. В боте выберите управление лицензиями - выберите нужный ключ - нажмите сменить HWID и отправьте боту тот, который вы получили в консоли
6. Перезапустите панель
7. Запустите ещё раз и подождите пока откроется интерфейс в браузере.
8. Если нужно, чтобы отображались стикеры и была возможность покупки прямо из панели, привяжите API ключ от маркета и перезапустите ещё раз панель

### Настройка API-ключа (очень важно!)

При первом запуске **стикеры не загрузятся** и покупка из панели будет недоступна.

**API-ключ нужен для:**
- Загрузки базы стикеров
- Покупки скинов прямо из интерфейса

**Если стикеры и покупка вам не нужны** — API-ключ можно не добавлять.

#### Как добавить API-ключ:
1. Перейдите по ссылке: [https://market.csgo.com/en/api/content/start#apigen]

<img width="1061" height="613" alt="Снимок экрана 2026-04-21 041819" src="https://github.com/user-attachments/assets/0551be83-3061-44aa-a7e2-2eafab80488c" />

2. Создайте и скопируйте свой ключ в панель
3. В панели нажмите кнопку **«Установить API-ключ»**.

<img width="860" height="638" alt="Снимок экрана 2026-04-21 042046" src="https://github.com/user-attachments/assets/ed03809e-1b31-4131-8a0a-cdd0a1681745" />


4. Вставьте ключ и нажмите **«Сохранить»**.
5. **Перезапустите** парсер.

После перезапуска в консоли появится сообщение, что база стикеров загружена.
