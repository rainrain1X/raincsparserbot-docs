# Добавление скинов

После того как вы выбрали или создали лист, нажмите кнопку **«Добавить скин»**.

Для каждого типа поиска есть отдельная вкладка:

### Поиск по Float
- Введите **точное название скина** на английском языке (как оно отображается на Market или в Steam Inventory Helper).
- Укажите нужный диапазон float.

### Поиск по Pattern
- Введите точное название скина.
- Добавьте необходимые паттерны через пробел.

### Поиск по стикерам
Есть два основных режима:

1. **Поиск только по комбинации стикеров** (по всем скинам).
2. **Поиск стикеров на конкретном скине** — поставьте галочку и введите название скина.

**Важно:**  
Названия стикеров вводите **только на английском**. Работает автокомплит — выбирайте именно английский вариант.

Скин добавится в выбранный лист и будет отображаться в разделе мониторинга.
