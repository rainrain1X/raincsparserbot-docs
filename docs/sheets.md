# Работа с листами скинов

При первом запуске парсера автоматически создаются стартовые листы.  
Вы можете их удалять и создавать свои собственные.

### Важно

Для каждого типа поиска используются **отдельные** листы:
- Поиск по Float
- Поиск по Pattern  
- Поиск по стикерам

Листы одного типа нельзя использовать в другом типе поиска.

### Как работать с листами

1. Создайте новый лист или выберите существующий.
2. Отметьте нужные листы галочкой — только из них будет происходить поиск.
3. Добавляйте в них скины.

Все добавленные скины будут отображаться в разделе **«Мониторинг»** (отдельная вкладка для каждого типа поиска).
