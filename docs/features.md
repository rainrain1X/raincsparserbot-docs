# Возможности парсера

Парсер RainCsParserBot позволяет искать скины тремя основными способами:

- **Поиск по Float** — точный диапазон износа
- **Поиск по Pattern** — конкретные паттерны (включая редкие)
- **Поиск по стикерам** — любые комбинации наклеек

### Дополнительные возможности

- Поиск **по брелкам**
- Одновременный запуск всех трёх типов поиска
- Создание и управление **несколькими листами скинов**
- Покупка скинов **прямо из интерфейса** (при наличии API-ключа)
- Отображение всех стикеров и брелоков на найденных скинах
- Сравнение цены с минимальной ценой на этот же скин

Вы можете создавать разные листы скинов и выбирать, по каким именно будет происходить поиск.
