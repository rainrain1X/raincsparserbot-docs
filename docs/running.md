# Запуск парсинга

После того как вы добавили скины и выбрали нужные листы, нажмите кнопку **«Запустить парсинг»**.

Поиск занимает в среднем **до 5 минут**.  
Когда кнопка снова станет зелёной — поиск завершён.

### Что вы увидите в результатах

В окне **«Найденные совпадения»** будут показаны **все** найденные скины:
- float / pattern / стикеры / брелки
- Текущая цена (зелёным)
- Минимальная цена на этот же скин (для удобного сравнения)

Каждый результат открывается в отдельной вкладке, даже если вы добавляли один и тот же скин с разными параметрами.
