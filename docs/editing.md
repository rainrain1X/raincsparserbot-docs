# Просмотр и редактирование скинов

Все добавленные скины отображаются в разделе **«Мониторинг»**.  
Для каждого типа поиска есть отдельная вкладка.

Показываются только те скины, листы которых вы отметили галочкой.

### Возможности редактирования

- **Удаление** ненужных скинов — прямо из списка.
- **Редактирование паттернов** — доступно только для поиска по Pattern (кнопка **Pattern** напротив скина).
- Стикеры отображаются в виде маленьких иконок. При наведении курсора появляется название стикера.

Здесь вы можете быстро проверить, что именно сейчас находится в работе, и при необходимости поправить параметры поиска.
