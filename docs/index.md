# Парсер для Market CS:GO 

**Добро пожаловать в документацию!**

Спасибо, что приобрели подписку на парсер CsParserBot.

Здесь вы найдёте всё, что нужно для быстрого и удобного использования парсера.

## Быстрый старт

1. Скачайте архив через бота(даст ссылку на канал с последним обновлением)
2. Распакуйте и запустите `.exe` файл
3. Добавьте API-ключ (если нужны стикеры и покупка из панели)
4. Готово! Можно начинать добавлять скины и запускать поиск.
